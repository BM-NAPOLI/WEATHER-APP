# Modern Weather App 🌤️

**Modern Weather App** is a sleek and responsive web application that shows the current weather for any city, including temperature, humidity, wind speed, pressure, and a 5-day forecast. Built with HTML, CSS, and JavaScript for a modern interactive interface.

---

## ⚡ Features

- Displays current weather with a large temperature and icon.
- Extra details: Humidity, Wind Speed, Pressure.
- 5-Day Weather Forecast.
- Responsive and visually appealing design with animated background.
- Search functionality ready to integrate with a weather API.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (with Google Fonts - Poppins)
- JavaScript (Vanilla JS)
- Animated gradient background effects

## 🚀 How to Run

1. Copy the project folder to your local machine.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge...).
3. You will see the weather UI with default data (Agadir, March 1, 2010).
4. You can modify the city or integrate a weather API to get live data.

---

## 📌 Notes

- The current data is **historical and estimated** (Agadir, March 1, 2010).
- Weather icons are emojis for simplicity and can be replaced with images or icon libraries.
- Current version uses static data; API integration can be added later for live weather updates.
